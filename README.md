## 🛠️ Technologies et Standards utilisés

Ce projet a été développé initialement, sans framework lourd, afin de garantir des performances optimales et une maîtrise totale du code :

* **HTML5** : Structure sémantique respectant les standards du web et l'accessibilité.
* **CSS3** : Design fluide (Responsive/Mobile-First) utilisant Flexbox, CSS Grid et des variables CSS.
* **Sécurité** : 
    * Implémentation d'une Content Security Policy (CSP) stricte.
    * Sécurisation des liens externes (`noopener noreferrer`).
    * Absence de backend dynamique pour réduire la surface d'attaque.
